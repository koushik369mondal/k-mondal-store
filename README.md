# K Mondal Store

A full-stack MERN e-commerce platform for a local kirana store.

##  Live Demo

- **Frontend**: [https://k-mondal-store-frontend.onrender.com](https://k-mondal-store-frontend.onrender.com)
- **Backend**: [https://k-mondal-store-backend.onrender.com](https://k-mondal-store-backend.onrender.com)

##  Quick Start

```bash
# Clone repository
git clone https://github.com/koushik369mondal/k-mondal-store.git

# Backend setup
cd backend
npm install
npm start

# Frontend setup
cd frontend
npm install
npm run dev
```

##  Features

- Product browsing and cart management
- User authentication & order tracking
- Admin dashboard for product & order management
- Premium UI with Tailwind CSS
- CI/CD pipeline with GitHub Actions
